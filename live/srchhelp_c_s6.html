Search by CCA New Material Library Sample ID Number:

Sample ID Number:
 

Type the New Material Library sample ID number, starting with the first numbers. By typing the first part of the sample ID number only, you can browse by New Material Library location. Replace decimal points with a space.

    Examples:
  • 01a 004
  • 01a
    Please Note: ID # designates the location of the material sample, and gives a sequential sample number. The materials library displays most samples on 24 vertical panels. Each panel is numbered, and its sides are designated "a" and "b". Each sample ID# begins with the panel location (e.g. 18a), then a decimal point, followed by a sequential sample number. Therefore the first sample on side a of panel 18 is numbered 18a.001.
